Top

The following are elements used when describing contents in the body element.
Do not forget to write document type declaration, html element, and head element.

Describe the structure of the document

h1 ~ h6

Represents that it is a heading.Do not use in a phrasing content (an element that do not constitute an independent block).

  • H1 is the highest, h6 is the lowest
  • Generally the size decreases in order from h1 to h6

p

Represents that it is a paragraph.Do not use in a phrasing content.

Apply styles to the text

b

Displays in bold style.

i

Displays in italic style.

strong

Emphasizes. Browser display becomes bold.

em

Emphasizes. Browser display becomes italic.

Itemization

ul

Combine with the li element to display unordered list.Do not use in a phrasing content.Child elements must be li elements.

ol

Combine with the li element to display ordered list.Do not use in a phrasing content.Child elements must be li elements.

li

Displays each item of list.

Linking

a

Displays the content of elements as links to another web page.

  • Specifies the URL of the linked page in its href attribute.

Image

img

Displays an image.Empty element.

  • Specifies the URL of the image to be displayed with its src attribute.

Table

table

Display the table by combining the tr, th and td. Do not use in a phrasing content.Child elements are tr and so on.

tr

Display one line in the table element. Child elements must be th or td elements.

th

Display the header cell in the tr element.

td

Display cell representing data in tr element.

Form

form

Display the form to transmit information to the web server. Do not use in a phrasing content.

button

Display a button.

input

Display controls for a form.

textarea

Display a multi-line text area.

select

Display a selection menu.Child elements must be option elements.

option

Display individual choices in a select element.

label

Display labels for controls on forms.

Others

hr

Display breaks between paragraphs.Empty element.Do not use in a phrasing content.

br

Perform a linefeed within a paragraph.Empty element.
